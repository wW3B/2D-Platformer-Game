About our game : เกมที่ทำเป็นเกมแนว 2D Platformer เป็นแนวคล้ายๆเกม Mario 
โดย Objective ของเกมจะมีการผ่านกับดักต่างๆ มี Coin,Gem และ Legendary Gem ให้เก็บ เพื่อเป็นคะแนนในการผ่านด่าน โดยแต่ละอันจะมีคะแนนที่ต่างกัน 

Why : เราเลือกทำเกมนี้เพราะว่าไม่ซับซ้อน และเข้าถึงง่ายกับทุกคน 

Third-Party program : โปรแกรมภายนอกที่ใช้จะมีสองโปรแกรม
- Tile Editor ใช้สำหรับการสร้าง Map ในเกม ซึ่งตัวโปรแกรมจะทำงายโดยการให่เรา import texture เข้าไปและวาดได้เลย รวมถึงสามารถกำหนดขอบเขตของ Block แต่ละ Block ได้ว่าต้องการให้เป็นอะไร เช่น กำหนดให้เป็นพื้นผิว player ก็จะไม่สามารถทะลุได้ หลังจากที่สร้างเสร็จเรียบร้อยก็ export ไปเก็บไว้ที่ Folder assets จากนั้น ก็เรียกใช้ฟังก์ชั่นของ badlogic เพื่อ import ไฟล์เข้ามา
- Texture Packer ใช้สำหรับการรวมไฟล์ sprite sheets หลายๆไฟล์ให้อยู่ในไฟล์เดียว และตั้งชื่อให้แต่ละ sprite เพื่อให้ง่ายต่อการเรียกใช้ โดยการใช้ฟังก์ชั่นของ badlogic เพื่อ import เข้ามาในเกมและระบุ ชื่อ sprite และตำแหน่งแกน x y รวมถึง index จากนั้นก็ใช้ร่วมกับ for loop เพื่อให้ขยับได้

Main menu : หน้านี้จะมีการ import รูป 5 รูป คือ 
main menu background = ใช้เป็นพื้นหลัง
logo = เป็นโลโก้ของเกม
press anykeys = เป็นการแสดงข้อความให้ player กดปุ่มอะไรก็ได้
exitidle = เป็นปุ่มออกเกมสีขาว
exithover = เป็นปุ่มออกเกมสีดำเมื่อเอา cursor ไปชี้

หน้านี้มีฟังก์ชั้นการใส่เพลงให้เข้ากับบรรยากาศเกมด้วยฟังก์ชั่น  Music
แล้วมีเสียงเมื่อ player ทำการกดปุ่มด้วยฟังก์ชั่น Sound
มีการทำให้คำว่า Press “ANYKAEYS” to start มีการกระพริบ



Control : เป็นหน้าต่อจาก main menu เมื่อ player กดปุ่นแล้วจะพามาหน้านี่เพื่อแสดงถึงปุ่มในเกมใช้เล่นเกม โดยฟังก์ชั่นหลักจะเป็นการเช็คปุ่มที่ player กด ถ้าผู้เล่นกดปุ่มที่ใช้งานถูกจะมีการแสดงให้เห็นว่าผู้เล่นกดถูกแล้วกดปุ่มใดอยู่ 
และมีการให้ผู้เล่นกด enter เพื่อ ไปยังเกมหลัก

Main Stage : Objective ของเกมจะมีการผ่านกับดักต่างๆ มี Coin,Gem และ Legendary Gem ให้เก็บ เพื่อเป็นคะแนนในการผ่านด่าน โดยแต่ละอันจะมีคะแนนที่ต่างกัน มีศัตรูในเกมโดย Player สามารถเลือกได้ว่าจะหนีหรือจะสู้ ถ้าสู้จะได้คะแนนพิเศษ โดยหน้านี้จะมีการดึงข้อมูลจากหลายๆ class เพื่อมาแสดง เช่น player , enemy , map ,coin , gem , legendarygem ,hud

โดยด้านบนจะมีการแสดง hud ต่างๆ เช่น 
- HP : จะแทนด้วยหัวใจสามดวง ซึ่งทำงานโดยการที่เราจะมรเซ็นเซอร์อยู่ที่ตัวผู้เล่น ถ้าเซ็นเซอร์นั้นไปโดนจุดที่เราระบุไว้ก็จะมีการลบรูปหัวใจไปตามที่กำหนด และเมื่อหัวใจหมดก็จะไปที่หน้า Gameover
- Score : จะเป็นการแสดงคะแนนของผู้เล่นจากการเก็บของหรือฆ่าศัตรู จะทำงานกับเซ็นเซอร์อีกเช่นกัน เมื่อเซ็นเซอร์ของผู้เล้นไปโดน Item ก็จะมีการเพิ่มคะแนนตามที่ระบุไว้
- Time : แสดงเวลานับถอยหลัง เมื่อเวลาหมดก็จะ Gameover
มีการใส่เสียงพื้นหลังเพื่อให้เข้ากับบรรยากาศพื้นหลังเกม มีเสียงในการกระโดดและเก็บไอเทมต่างๆ และมีฟังก์ชันการทำลาย block บาง block ได้
